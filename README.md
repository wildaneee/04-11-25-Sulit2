# 04-11-25-Sulit2

Penjelasan Program:
Pengguna memasukkan ukuran matriks (baris dan kolom).

Program meminta input elemen-elemen dari dua matriks: Matriks A dan Matriks B.

Penjumlahan dilakukan elemen per elemen: hasil[i][j] = A[i][j] + B[i][j].

Hasil penjumlahan ditampilkan dalam bentuk matriks.
